В репозитории располагается решение следующей задачи:

Написать программу, которая из имеющегося массива строк формирует 
новый массив из строк, длина которых меньше, либо равна 3 символам. 
Первоначальный массив можно ввести с клавиатуры, либо задать на старте 
выполнения алгоритма. При решении не рекомендуется пользоваться коллекциями, 
лучше обойтись исключительно массивами.

Задание прямо не указывает, на каком языке программирования нужна
реализация задачи. В этой связи она осуществлена на языках:
С# (файл Csharp.cs)
Python (файл Python.py)

При этом, решение в С# можно назвать основным, подробным. В то время как решение
в Python представляет собой краткую альтернативу.

Также, репозиторий содержит визуализацию программы на С# в виде блок схем 
двух методов (файл Block_diagram.png).
По существу, схема описывает и происходящее "под капотом" в программе на языке Python.

Вкратце, реализация сводится к запросу у пользователя строки, которая через
пробельный разделитель содержит "слова". Эта строка разбивается по пробелу 
на слова и записывается в массив. Далее, определяется количество слов, 
символьно меньших 3. Определяется новый массив, равный количеству таких элементов.
Происходит запись элементов в этот массив.
В финале, дополнительно (что не указано заданием) и для наглядности, массив выводится
поэлементно в консоль.

Инструкция:
После запроса со стороны программы, следует ввести строку.
Она будет разбита по пробелам и возвращена в составе переменных
 shortWords (С#) или short_words (Python), а также выведена в консоль.
